**Create src/styles/typography.css** - Fluid typography rules:

Base typography (mobile-first):
```css
body {
  font-family: var(--font-sans);
  font-size: 1rem; /* 16px baseline */
  line-height: 1.6;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

/* Tablet+ increase for better readability */
@media (min-width: 768px) {
  body {
    font-size: 1.0625rem; /* 17px */
  }
}
```

Fluid headings using clamp() with rem + vw for zoom support:
```css
h1 {
  font-size: clamp(2rem, 5vw + 1rem, 3.5rem);
  line-height: 1.2;
  font-weight: 600;
  letter-spacing: -0.02em;
  margin: 0 0 var(--space-md);
}

h2 {
  font-size: clamp(1.5rem, 4vw + 0.5rem, 2.5rem);
  line-height: 1.3;
  font-weight: 600;
  margin: 0 0 var(--space-sm);
}

h3 {
  font-size: clamp(1.25rem, 3vw + 0.5rem, 1.875rem);
  line-height: 1.4;
  font-weight: 500;
  margin: 0 0 var(--space-sm);
}
```

Paragraph and text:
```css
p {
  font-size: 1rem;
  line-height: 1.7;
  margin: 0 0 var(--space-md);
}

/* Large intro text for hero sections */
.text-lg {
  font-size: clamp(1rem, 2.5vw, 1.25rem);
  line-height: 1.8;
}

/* Small utility text */
.text-sm {
  font-size: var(--font-size-sm);
  line-height: 1.5;
}
```

Word break safety for long words:
```css
h1, h2, h3, h4, h5, h6 {
  word-wrap: break-word;
  overflow-wrap: break-word;
  hyphens: auto;
}
```
